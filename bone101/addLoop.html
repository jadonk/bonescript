Bonescript: addLoop

This is a prototype of a function to add additional running loops. It will eventually be cleaned up a bit, but you can get started with it today.

led = bone.USR3;
pinMode(led, 'out');

blinkLED = function() {
  digitalWrite(bone.USR3, 1);
  delay(100);
  digitalWrite(bone.USR3, 0);
  delay(100);
};

addLoop(blinkLED);
